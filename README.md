by selenium



## 使用方法

1.打开python文件把以下内容换成自己的：

```{python}
# 学号
id_card = '1901xxxxx'
# 密码
pswd = 'pswd'
# 出校事由
chu_shiyou = '玩'
# 出校轨迹
chu_guiji = '南门'
# 入校事由
ru_shiyou = '回宿舍'
# 入校轨迹
ru_jiedao = '北京大学'
```

2.安装selenium

```{python}
pip install selenium
```

3.运行

有自动弹框 不用管

打印出success说明成功备案
